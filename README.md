# 5-Stage RV32I Pipelined Processor

## Features
- 5-stage pipeline (IF, ID, EX, MEM, WB)
- Hazard Detection Unit
- Forwarding Unit
- Stall & Flush Control
- Functionally verified in simulation

## Tools
- Verilog HDL
- Xilinx Vivado (Simulation)
- Ubuntu 24.04
